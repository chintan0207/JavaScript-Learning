Deep & Shallow copy

When to Use Each

Shallow Copy: Use it when you know the object does not have nested objects or you don't need to preserve the immutability of nested structures.

  1. by using Object.assign()
  2. by using spread operator

Deep Copy: Use it when you need a complete clone of the object, ensuring changes to the copy do not affect the original object and vice versa.

  1. by using JSON.parse(JSON.stringify(origanal))
    1. Note: This method has limitations (e.g., it can't copy functions, undefined, NaN, Infinity, or special object types like Date, Map, Set, etc.)
  2. by using library _.cloneDeep(original)
